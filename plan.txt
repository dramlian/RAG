  qdrant

   docker run -d   --name qdrant   -p 6333:6333   -v qdrant_storage:/qdrant/storage   qdrant/qdrant


2. Create collection
curl -X PUT "http://localhost:6333/collections/mycollection" \
  -H "Content-Type: application/json" \
  -d '{
    "vectors": {"size": 768, "distance": "Cosine"}
  }'


For small tests, you can set size: 3 or any dimension.

3. Insert points
curl -X POST "http://localhost:6333/collections/mycollection/points" \
  -H "Content-Type: application/json" \
  -d '{
    "points": [
      {
        "id": 1,
        "vector": [0.1, 0.5, -0.2],
        "payload": {"text":"Hello world", "category":"example"}
      }
    ]
  }'


vector → match collection size

payload → your metadata

id → unique identifier

4. Search by vector
curl -X POST "http://localhost:6333/collections/mycollection/points/search" \
  -H "Content-Type: application/json" \
  -d '{
    "vector": [0.1, 0.5, -0.2],
    "limit": 10,
    "with_payload": true
  }'


Returns nearest points with payload and score.

5. List collections
curl http://localhost:6333/collections

6. Get collection details
curl http://localhost:6333/collections/mycollection

7. Delete collection
curl -X DELETE "http://localhost:6333/collections/mycollection"
